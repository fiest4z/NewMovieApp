# Настройка TMDB API

Для работы функции поиска фильмов необходимо получить бесплатный API ключ от The Movie Database (TMDB).

## Шаг 1: Регистрация на TMDB

1. Перейдите на сайт: https://www.themoviedb.org/
2. Нажмите "Sign Up" (Регистрация)
3. Заполните форму регистрации
4. Подтвердите email

## Шаг 2: Получение API ключа

1. Войдите в свой аккаунт
2. Перейдите в настройки: https://www.themoviedb.org/settings/api
3. Нажмите "Request API Key"
4. Выберите "Developer" (Разработчик)
5. Заполните форму:
   - Application Name: Movie Watchlist (или любое другое название)
   - Application URL: можно оставить пустым или указать свой сайт
   - Application Summary: опишите ваше приложение
6. Примите условия использования
7. Нажмите "Submit"

## Шаг 3: Добавление API ключа в приложение

1. Откройте файл `lib/services/tmdb_service.dart`
2. Найдите строку:
   ```dart
   static const String _apiKey = 'YOUR_TMDB_API_KEY_HERE';
   ```
3. Замените `YOUR_TMDB_API_KEY_HERE` на ваш API ключ из TMDB
4. Сохраните файл

## Пример:

```dart
static const String _apiKey = '1234567890abcdef1234567890abcdef';
```

## Готово!

После добавления API ключа функция поиска фильмов будет работать. Вы сможете:
- Искать фильмы по названию
- Получать информацию о фильмах из базы TMDB
- Видеть постеры фильмов
- Получать рейтинги, описания и другую информацию

## Примечания

- API ключ бесплатный и не имеет ограничений для личного использования
- Не публикуйте ваш API ключ в публичных репозиториях
- Если вы используете Git, добавьте файл с API ключом в `.gitignore`

## Альтернатива: Использование переменных окружения

Для большей безопасности можно использовать переменные окружения или файл конфигурации, который не попадает в Git.
