# Настройка OMDb API

Для работы функции поиска фильмов необходимо получить бесплатный API ключ от OMDb (Open Movie Database).

## Что такое OMDb?

OMDb (Open Movie Database) - это бесплатный веб-сервис для получения информации о фильмах. Он использует данные из IMDB и других источников.

## Шаг 1: Получение API ключа

1. Перейдите на сайт: http://www.omdbapi.com/apikey.aspx
2. Выберите тип ключа:
   - **FREE** - бесплатный ключ (1000 запросов в день)
   - **Patreon** - платный ключ (больше запросов)
3. Введите ваш email адрес
4. Подтвердите email (проверьте почту)
5. Получите ваш API ключ

## Шаг 2: Добавление API ключа в приложение

1. Откройте файл `lib/services/omdb_service.dart`
2. Найдите строку:
   ```dart
   static const String _apiKey = 'YOUR_OMDB_API_KEY_HERE';
   ```
3. Замените `YOUR_OMDB_API_KEY_HERE` на ваш API ключ из OMDb
4. Сохраните файл

## Пример:

```dart
static const String _apiKey = '12345678';
```

## Ограничения бесплатного ключа

- **1000 запросов в день** - достаточно для личного использования
- Если нужно больше, рассмотрите платный вариант

## Преимущества OMDb API

- ✅ Бесплатный план доступен
- ✅ Данные из IMDB
- ✅ Простой API
- ✅ Поддержка постеров
- ✅ Детальная информация о фильмах

## Готово!

После добавления API ключа функция поиска фильмов будет работать. Вы сможете:
- Искать фильмы по названию
- Получать информацию о фильмах из базы OMDb/IMDB
- Видеть постеры фильмов
- Получать рейтинги, описания и другую информацию

## Примечания

- API ключ бесплатный для личного использования
- Не публикуйте ваш API ключ в публичных репозиториях
- Если вы используете Git, добавьте файл с API ключом в `.gitignore`

## Альтернатива: Использование переменных окружения

Для большей безопасности можно использовать переменные окружения или файл конфигурации, который не попадает в Git.
